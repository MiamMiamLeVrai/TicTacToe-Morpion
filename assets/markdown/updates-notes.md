# Tic Tac Toe - Morpion

## 🇺🇸 Update Notes

### ⚙️ NEW VERSION: Version 1.2.0 (2026/02/27)

#### What's new 👀?

##### HTML Files

1. For `index.html`, minor corrections to the file, particularly on the choice of available languages, to better fit screen readers and HTML semantics,
2. Minors corrections in some of the HTML files,
3. Updated the version number from 1.1.1 to 1.2.0.

##### CSS3 Files (SCSS)

1. For `index.scss`, light reworks on the design of the choices of available languages, allowing better readability and more easily adding other available languages in the next updates,
2. For `homepage.scss`, light layouts of some elements and HTML design for better readability,
3. For `download.scss`, additions and corrections of features for screen readers and keyboard navigation,
4. Minor corrections in the SCSS files.

##### JavaScript Files

1. For `animation.js`, new animations for some buttons on the site, including the navigation buttons "Go to content" and "Go to Get elements",
2. Rewriting the code to better adapt to multiple HTML files,
3. For `tictactoe-en.js`, added new animations for some elements of the game.

##### Others Things

1. The `draft` file has been removed, as it was not used and did not add value to the site,
2. Minor corrections in the `README.md` file,
3. Minor corrections in the `site.webmanifest` file,
4. Minor additions in the `.gitignore` and `robots.txt` files,
5. `updates-notes.md` will only present new versions for better understanding.
